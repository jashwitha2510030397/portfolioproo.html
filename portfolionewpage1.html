Description about this project

I created this songs webpage to design a simple online music platform interface. The page displays Tollywood love songs with details such as the song name, movie name, and options to play, add to the library, or download the song. I used HTML and CSS to design the layout, navigation menu, and buttons to make the webpage look organized and user-friendly. This project helped me practice creating a basic music streaming style webpage.
